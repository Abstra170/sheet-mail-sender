What is Sheet Mail Sender?

Sheet Mail Sender helps users send personalized emails using Gmail drafts and Google Sheets data. Users can manage email campaigns, replace dynamic tags, send emails row by row, and track sending status.

Features

  • Send personalized Gmail emails from Google Sheets
  • Use Gmail drafts as email templates
  • Replace dynamic tags like {{Name}}, {{Company}}, and {{Email}}
  • Track sent, failed, and error status inside the sheet
  • User-controlled sending through their own Google account
